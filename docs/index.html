How it works

1

Session

Pick any Claude Code session. Postmaxxing reads the conversation, not the diff.

2

Extract

Haiku finds the decisions, trade-offs, and numbers. Sonnet turns them into posts.

3

Post

Edit inline. Hit Post to X. One click opens a pre-filled tweet composer.

Get running in 2 minutes

Zero npm installs. Just Node.js and an API key.

01

Download

Download the ZIP or git clone https://github.com/myrons/postmaxxing

02

Add your API key

Create a .env file in the folder: ANTHROPIC_API_KEY=sk-ant-...

Get a key at console.anthropic.com · costs ~$0.01/run

03

Run it

node devlog.js --ui — browser opens automatically, sessions load, start generating.

Built in public using itself. Every post about building this was generated by this.
